# 数据库配置指南

## 🔧 问题原因

错误信息：`Access denied for user 'root'@'localhost' (using password: NO)`

**原因**：
1. 数据库未配置密码
2. `.env` 文件中的 `DB_PASSWORD` 未正确设置

---

## ✅ 解决方案

### 方案一：使用空密码（本地开发推荐）

#### 1. 修改 .env 文件

```bash
# 将 DB_PASSWORD 设置为空
DB_PASSWORD=
```

#### 2. 确认MySQL root用户无密码

```bash
# 测试连接
mysql -u root
```

#### 3. 创建数据库

```sql
CREATE DATABASE emergency_dispatch CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 4. 运行迁移

```bash
npm run db:migrate
npm run db:seed
```

---

### 方案二：设置密码

#### 1. 修改 .env 文件

```bash
# 设置你的MySQL root密码
DB_PASSWORD=YourMySQLPassword
```

#### 2. 创建数据库

```sql
CREATE DATABASE emergency_dispatch CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 3. 运行迁移

```bash
npm run db:migrate
npm run db:seed
```

## 📝 完整配置步骤

### 步骤1：确认MySQL运行中

```bash
# Windows
net start MySQL

# Linux/Mac
sudo service mysql start
# 或
sudo systemctl start mysql
```

### 步骤2：配置 .env 文件

编辑 `backend/.env`：

```bash
# 如果数据库root用户有密码
DB_PASSWORD=your_password

# 如果数据库root用户无密码（本地开发）
DB_PASSWORD=
```

### 步骤3：创建数据库

```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE emergency_dispatch CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 退出
exit;
```

### 步骤4：运行迁移脚本

```bash
cd backend

# 迁移表结构
npm run db:migrate

# 初始化数据
npm run db:seed
```

## 🔍 常见问题

### Q1: MySQL服务未启动

**错误**: `Can't connect to MySQL server on 'localhost:3306'`

**解决**:
```bash
# 启动MySQL服务
# Windows: 在服务管理器中启动MySQL
# Linux: sudo service mysql start
# Mac: brew services start mysql
```

### Q2: root密码忘记

**解决**:
```bash
# 停止MySQL服务
# Windows: net stop MySQL
# Linux: sudo service mysql stop

# 以安全模式启动MySQL
mysqld --skip-grant-tables &

# 登录（无密码）
mysql -u root

# 重置密码
FLUSH PRIVILEGES;
ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';
```

### Q3: 端口3306被占用

**解决**: 检查MySQL配置并使用正确端口

## 🚀 快速启动（推荐配置）

### 1. 使用无密码配置（最简单）

编辑 `.env`：
```env
DB_HOST=localhost
DB_PORT=3306
DB_NAME=emergency_dispatch
DB_USER=root
DB_PASSWORD=
```

### 2. 创建数据库并运行

```sql
CREATE DATABASE emergency_dispatch CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

```bash
npm run db:migrate
npm run db:seed
npm run dev
```
